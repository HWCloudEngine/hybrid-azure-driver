3. 创建云使用用户和密码:
    - [参考这里](https://azure.microsoft.com/en-us/documentation/articles/active-directory-create-users/)

    - AZURE_SUBSCRIPTION_ID USER PASSWORD三个信息可以通过接口管理azure资源.
